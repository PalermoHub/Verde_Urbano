Esempio di Pagina con Header e Footer Designer Italia

Questa pagina dimostra l'implementazione di Header e Footer conformi alle linee guida di Designer Italia per i siti web dei Comuni italiani.

Conformità alle Linee Guida

Il design è conforme alle linee guida AgID e ai requisiti previsti dal PNRR per la digitalizzazione della Pubblica Amministrazione.

Caratteristiche dell'Header

L'header è strutturato in tre livelli come richiesto dalle linee guida:

  • Slim Header: Contiene il collegamento alla Regione Siciliana, selezione lingua e accesso all'area personale
  • Header Centrale: Logo del Comune, denominazione dell'ente, collegamenti social e funzione di ricerca
  • Header di Navigazione: Menu principale con le sezioni del sito, responsive e accessibile
  • Banner Progetto: Titolo del progetto con link al CUP OpenCUP

Caratteristiche del Footer

Il footer include tutte le informazioni obbligatorie richieste dalla normativa:

  • Informazioni di contatto: Indirizzo, PEC, telefono, codice fiscale e P.IVA
  • Link normativi obbligatori: Amministrazione Trasparente, Privacy, Note Legali, Dichiarazione di Accessibilità
  • Servizi al cittadino: Prenotazione appuntamenti, assistenza, segnalazioni, FAQ
  • Social network: Collegamenti ai canali social ufficiali del Comune
  • Credits e riferimenti PNRR: Informazioni sul progetto, Designer Italia, Bootstrap Italia

Tecnologie Utilizzate

  • Bootstrap Italia: Framework CSS conforme alle linee guida
  • Titillium Web: Font ufficiale per i siti della PA
  • Accessibilità WCAG 2.1: Conforme agli standard di accessibilità
  • Design Responsive: Ottimizzato per tutti i dispositivi

File Creati

I seguenti file sono stati creati per implementare il design conforme a Designer Italia:

  • includes/header.html - Header completo a tre livelli
  • includes/footer-designer-italia.html - Footer conforme alle linee guida
  • css/designer-italia-custom.css - Stili personalizzati conformi
  • esempio-designer-italia.html - Questa pagina di esempio

Bootstrap Italia

Per utilizzare pienamente le funzionalità, è necessario scaricare Bootstrap Italia da GitHub e includere i file CSS e JavaScript nel progetto. Le icone SVG ufficiali sono incluse nel pacchetto Bootstrap Italia.

Prossimi Passi

  1. Scaricare Bootstrap Italia dal repository ufficiale
  2. Integrare gli sprites SVG per le icone ufficiali
  3. Aggiornare i link del footer con quelli effettivi del Comune
  4. Personalizzare i colori mantenendo la conformità alle linee guida
  5. Testare l'accessibilità con strumenti di validazione WCAG
